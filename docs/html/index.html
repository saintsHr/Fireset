Fireset

Introduction

Fireset is a game engine written in C, its main propouse is to make game
development more enjoyable and easy, and also give your games that old feel
we all love, in this documentation you have all functions, types and structs
documented in detail and as well some step by step guides!

Installing

Debian:

  • go to https://github.com/saintsHr/Fireset
  • go to Releases tab and install the latest stable release
  • install your .deb:
    sudo apt install ./fireset-dev_<version>_amd64.deb
  • now use those flags in your compilation:
    -Ifireset -lfireset -lglfw -lGL
    or:
    $(pkg-config --cflags --libs fireset)
  • and this line in your source files:
    #include <fireset/fireset.h>
  • done, now you're ready to go!

Manual:

  • go to https://github.com/saintsHr/Fireset
  • go to Releases tab and install the latest stable release
  • extract your .zip
  • now you should see the folder lib and also include
  • move those to any place you like
  • now use those flags in your compilation:
    -Lpath/to/lib/folder -lfireset -Ifireset -lfireset -lglfw -lGL -lm -Ipath/to/include/folder
  • if you get any linking errors related to OpenGL or GLFW, make sure you have it installed and also add a "-L" flag for those
  • and this line in your source files:
    #include <fireset/fireset.h>
  • done, now you're ready to go!

Compiling

This guide is for those who want to compile the engine by hand, we do not recommend
doing this for normal usage, anyways, heres the step-by-step guide:

  • Install the dependencies:
    • OpenGL Immediate Mode (3.3 or less)
    • GLFW
  • Go to https://github.com/saintsHr/Fireset
  • Press de "Code" button
  • Click "Download ZIP"
  • Extract it
  • Now you should see some folders, like:
    • src, source files (.c)
    • include, header files (.h)
    • etc.
  • Open your terminal in that folder
  • for a debug version, run:
    make
  • for a release version, run:
    make release
  • to compile the docs, run:
    make doc
  • to compile the .deb, run:
    make deb
  • to remove build files, run:
    make clean
  • now you should see 3 extra folders:
    • bin: contains the test executable
    • build: contains the object (.o) and dependencies files (.d)
    • lib: contains the compiled engine (.a)
  • done, you have compiled the engine!

Quickstart

This guide is for those who want to have a window as fast as possible,
or for those who want to take a quick look on the API ,so heres a simple example:

#include <fireset/fireset.h>
int main(void){
// inits engine
if (!fsInit()) return 1;
// creates window
FsWindow window;
window.width = 800;
window.height = 600;
window.name = "Example Game";
fsCreateWindow(&window);
// main loop
while (!fsWindowShouldClose(window)){
fsClear((FsColor){0, 0, 0});
// render here
fsHandleWindow(&window);
}
// destroy window and closes engine
fsDestroyWindow(&window);
fsExit();
return 0;
}
